# 🎯 Hangman Game with SFML

![C++](https://img.shields.io/badge/C%2B%2B-20-blue)
![SFML](https://img.shields.io/badge/SFML-3.1-green)
![CMake](https://img.shields.io/badge/CMake-3.28+-red)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

A complete Hangman game built with **C++20**, **SFML 3.1**, and **CMake**.

The game features multiple difficulty levels, custom medieval-themed artwork, audio controls, pause functionality, and a polished user interface.

## Highlights

- Built with C++20 and SFML 3.1
- Custom medieval-themed UI and artwork
- Multiple difficulty levels
- Audio settings saved to JSON
- Pause menu and options menu
- State-based game architecture

# 🔨 Building

## Clone Repository

```bash
git clone https://github.com/KrissooT/Hangman-Game-with-SFML.git
cd Hangman-Game-with-SFML
```

## Configure

```bash
cmake -B build
```

## Build

```bash
cmake --build build
```

## Run

Launch the generated executable from the build directory.
